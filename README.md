**Reflection**:

The biggest challenge I faced was getting everything set up since it was my first time using these tools. One of the more confusing parts was wondering why I kept getting internal server errors when making API calls. I then realized that it was because I didn’t update the permission of the lambda to write to dynamoDB. One of the things I learned while working on this assignment is that since AWS lambda does not have any servers to manage, I can focus on writing code instead. It’s convenient since it’s one less thing to think about. I also learned about using DynamoDB and that it is a NoSQL database. I like how there is schema flexibility.
